# 🚀 Blog Application

A full-stack blogging platform built using Node.js, Express, and EJS. This application allows users to create, read, update, and delete blog posts with a simple and clean interface.

---

## 📌 Features

* 📝 Create, edit, and delete blog posts
* 📖 View all blogs and individual posts
* ⚡ Server-side rendering using EJS
* 🎯 Simple and clean UI
* 🗂️ Organized project structure

---

## 🛠️ Tech Stack

* **Backend:** Node.js, Express.js
* **Frontend:** EJS, HTML, CSS
* **Database:** MongoDB / PostgreSQL (update as per your project)
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

### 1. Clone the repository

git clone [https://github.com/skgnandy/Blog.git](https://github.com/skgnandy/Blog.git)
cd Blog

### 2. Install dependencies

npm install

### 3. Run the application

npm start

### 4. Open in browser

[http://localhost:3000](http://localhost:3000)

---

## 📌 Routes

| Method | Route       | Description     |
| ------ | ----------- | --------------- |
| GET    | /           | Get all blogs   |
| GET    | /post/:id   | Get single blog |
| POST   | /create     | Create new blog |
| PUT    | /edit/:id   | Update blog     |
| DELETE | /delete/:id | Delete blog     |
